# Bag Dictionary Project

This project was created for my CSIS 215 Data Structures course. The goal of the project was to build a bag-based dictionary in C++ using templates, inheritance, abstract data types (ADTs), and dynamic memory allocation.

The program uses an array-based bag implementation to store key-value pairs and perform common dictionary operations such as inserting, removing, finding, and clearing records.

## Features

* Array-based bag implementation
* Dictionary built using key-value pairs
* Insert, remove, find, and clear functions
* Template support for multiple data types
* Object-oriented programming concepts
* Dynamic memory allocation

## Files Included

* `ABag.h` – Array-based bag implementation
* `BDictionary.h` – Dictionary implementation using the bag
* `bagADT.h` – Bag abstract data type
* `dictionaryADT.h` – Dictionary abstract data type
* `kvpair.h` – Key-value pair class
* `bagtestmain.cpp` – Main testing file

## Technologies Used

* C++
* Visual Studio
* Templates
* Inheritance
* ADTs
* Dynamic memory

## Example Output

The program demonstrates:

* inserting records
* removing records
* finding records
* clearing dictionaries
* using multiple template data types

This project helped me better understand how abstract data types work and how templates can be used to create reusable code. I also got more practice working with dynamic memory, inheritance, and object-oriented programming in C++.
